# UMELISA-TSH-Neonatal
Software para procesamiento de curvas de calibración en ensayos UMELISA TSH Neonatal. Implementa nueve métodos: lineal punto a punto, semilog piecewise, polinomios grado 2 y 3, splines cúbicos, spline Akima y modelos 4PL/5PL. Evalúa precisión y estabilidad para pesquisaje de Hipotiroidismo Congénito. Desarrollado con Python/Tkinter.
